# Иконки расширения

Для работы расширения нужны PNG иконки:
- `icon16.png` (16x16)
- `icon48.png` (48x48)
- `icon128.png` (128x128)

## Как создать

1. Используй SVG шаблон из `icon16.svg`
2. Экспортируй в нужные размеры через:
   - Figma
   - https://www.iloveimg.com/resize-image
   - Photoshop/GIMP

## Дизайн
- Фон: #6366f1 (индиго)
- Иконка: белая звезда
- Скруглённые углы

## Временное решение
Для тестирования можно использовать любые PNG-картинки 
указанных размеров.
