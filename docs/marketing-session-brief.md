## Что делает маркетинговая сессия

### 1. TG-бот для контента
- Постинг текстов в канал через Telegram Bot API
- Генерация контента (советы, кейсы, факты по работе с отзывами)
- Расписание: Пн (совет), Ср (отзыв недели), Пт (факт/статистика)
- Карусели/картинки через Freepik API (подключён как MCP)

### 2. Контент для запуска
- Тексты постов для личных соцсетей Сергея (Stage 1)
- Тексты постов для Edison Bar каналов (Stage 2)
- Черновик статьи для VC.ru (Stage 3)
- Контент для TG-канала MyReply — первые 10-15 постов на посев

### 3. Промокоды
Создать в Supabase (через MCP):
- **ЛЯГУШКА** — 7 дней Start (личные каналы)
- **EDISON** — 7 дней Start (аудитория бара)
- **VCRU** — 3 дня Start (VC.ru запуск)
